---- INSTRUCTIONS TO COMPILE USING THE DELTA COMPILER ----
- DO NOT erase the results folder, this is where the resultant .wat codes will store.
- Run from this root directory the compile.py Python code and wait, to see further information have an open terminal.
- Check the resultant codes in the results directory.
- If you want to add more sourde .js codes add them to the dataset directory, make sure they have the structure, blocks and language rules declared in the delta/delta.peg file.


---- INSTRUCTIONS TO EXECUTE THE RESULTANT WASM CODE ----
- Run the run_wasm.py once you have compiled all your .js source codes into WASM files.
- To see further information have an open terminal.
